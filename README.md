# SwiftyLISP REPL

**A basic REPL for SwiftyLISP**

## Summary

This is a simple REPL for the Swift LISP interpreter described in [this article](https://www.uraimo.com/2017/02/06/building-a-lisp-from-scratch-with-swift/).

## Usage

Compile and run.
